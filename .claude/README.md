# Claude Code Integration

This folder contains resources and instructions tailored for Claude to assist with the `sap-btp-cloud-logging-client` usage.

## Folder Structure

- **instructions**: Detailed prompts and guides for Claude to execute specific tasks.
    - `setup-logging.md`: Guide to setting up the library in a new project.
- **code**: Source code templates for reuse.
    - `LogUtils.ts`: Standard TypeScript utility class for consistent logging.

## Usage

When using Claude in this project or related contexts, you can point it to `setup-logging.md` to quickly understand the best practices for integration.
